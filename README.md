The Blockchain Product Authentication System is a secure and efficient solution designed to prevent counterfeit products using blockchain technology. This project integrates smart contracts, QR code generation, and a user-friendly web interface to ensure product authenticity and transparency in supply chains. Manufacturers can register products on a blockchain network, where the data is stored in an immutable and tamper-proof manner. Each product is assigned a unique QR code, which can be generated and shared with consumers. By scanning the QR code, users can instantly verify whether a product is genuine or fake through real-time blockchain validation.

The system is developed using Python and Streamlit for the frontend interface, along with Web3.py and Ganache for blockchain interaction and simulation. A Solidity smart contract is used to manage product registration and verification, ensuring that only authorized users can add products and preventing duplicate entries. Additionally, technologies like OpenCV, Pyzbar, and PIL are used for QR code scanning and image processing. The application provides three main functionalities: product registration, QR code generation, and product verification, making it a complete end-to-end authentication system.
